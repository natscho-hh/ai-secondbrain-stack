# Writing style

The rules your agent follows whenever it writes in your voice — posts, emails, docs. This file beats any generic style the agent would default to. Replace the italicized examples with your own rules; add to the lists over time whenever you catch the agent phrasing something in a way you'd never say.

## Tone

*Direct and informal, like explaining something to a smart colleague. Confident but never salesy.*

## Address

*First names, informal "you" — never "Dear Sir or Madam".*

## Hard rules

*- Short sentences. One idea per sentence where possible.*
*- No emojis in professional writing.*
*- Numbers as digits (3, not "three").*

## Avoid

*Corporate jargon ("synergy", "leverage"), hype words ("game-changing", "revolutionary"), and filler openers ("In today's fast-paced world…").*
