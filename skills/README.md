# Skills

A skill is a self-contained folder with one job and a `SKILL.md` file that tells an AI agent exactly how to do it — when to use it, what steps to follow, and what rules never to break. Each skill lives in its own subfolder here (`skills/<skill-name>/SKILL.md`), so skills stay portable and easy to add, remove, or share independently. Claude Code loads skills from this folder natively; any other agent (Codex, Gemini CLI, OpenCode) uses them the same way manually — list this folder, find the `SKILL.md` that matches the task, read it, and follow its steps in order. For the full curated catalog of skills — core and community — see `manifest/skills.md`.
