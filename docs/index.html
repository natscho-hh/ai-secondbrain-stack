AI SecondBrain OS

A git-versioned second brain that works with ANY AI coding agent — Claude Code, Codex, Gemini CLI, Cursor, Copilot. Switch agents anytime; they all follow the same rules.

Claude Code

  1. Install Claude Code: curl -fsSL https://claude.ai/install.sh | bash — see the official docs.
  2. Create a new, dedicated, empty folder for your second brain and open it in your terminal. Don't point it at an existing project — give it a clean home of its own.
  3. Launch claude in that folder, then copy the Bootstrap Prompt below and paste it in.

    
    
Your agent takes over from here.
